# Intro to TDD and RSpec

## Instructions

Clone this repository:
```
$ mkdir tdd_pairing
$ git clone git@github.com:darciew/tdd_pairing.git
$ cd tdd_pairing
```

Install bundle by running:
```
$ gem install bundle
```

Instal required gems by running:
```
$ bundle
```

To run rspec tests from the directory:
```
$ rspec
```
